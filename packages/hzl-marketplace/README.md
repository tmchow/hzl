# HZL Marketplace

Official Claude Code marketplace for HZL plugins and skills.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add tmchow/hzl
```

## Available Plugins

### hzl-skills

Skills for AI agents using HZL task tracking.

```bash
/plugin install hzl-skills@hzl-marketplace
```
